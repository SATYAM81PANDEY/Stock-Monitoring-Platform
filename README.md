# Zerodha Clone

This is a stock monitoring platform clone project.  
It includes:
- Backend (Node.js + Express + MongoDB)
- Frontend (React)
- Dashboard (React)

## Installation

1. Clone the repo
2. Run `npm install` in backend, frontend, and dashboard
3. Run `npm start` for each project

## Features

- User authentication
- Real-time stock updates
- Portfolio management
